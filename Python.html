1. Introdução ao Python

Python é uma linguagem simples e poderosa.

Exemplo:

print("Olá, Mundo!")

Atividade: Imprima seu nome

2. Variáveis e Tipos

Exemplo:

nome = "João"
idade = 25
print(f"{nome} tem {idade} anos.")

Atividade: Crie suas variáveis

3. Estruturas de Controle (Loop for)

Exemplo de loop:

for i in range(1, 6):
    print(i)

Atividade: Imprima de 1 a 10

4. Funções

Exemplo:

def soma(a, b):
    return a + b

print(soma(3, 5))

Atividade: Função de saudação

5. Listas

Exemplo:

frutas = ["maçã", "banana"]
frutas.append("laranja")
print(frutas)

Atividade: Sua lista

6. Manipulação de Arquivos (Simulado)

No navegador, simulamos com prints.

conteudo = "Olá!"
print("Escrevendo no arquivo:", conteudo)

Atividade: Simule escrita

7. Orientação a Objetos

Exemplo de classe:

class Pessoa:
    def __init__(self, nome):
        self.nome = nome
    
    def falar(self):
        print(f"Olá, sou {self.nome}")

p = Pessoa("Ana")
p.falar()

Atividade: Classe Carro

8. Módulos (math)

Exemplo:

import math
print(math.sqrt(16))

Atividade: Use math
